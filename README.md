# Sporting Goods E-Commerce Website

A full-stack e-commerce web application for a sporting goods store, built with JS, HTML/CSS, PHP, and MySQL.

## Features
- Product browsing by sport category
- User registration and login
- Shopping cart and checkout functionality
- Admin panel for stock management

## Technologies
- PHP
- MySQL
- HTML / CSS / JavaScript
